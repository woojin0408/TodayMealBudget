# Feature Spec

## Home

### 목적

오늘 내가 집중해서 벌어둔 식비 보너스를 한눈에 보여준다.

### 표시 요소

- 앱 이름
- 오늘 날짜
- 점심 예산 카드
- 저녁 예산 카드
- 오늘 총 집중 시간
- 오늘 획득 보너스
- 집중 시작 버튼
- 메뉴 추천 버튼

### 계산

```text
점심 총 예산 = 점심 기본 식비 + 오늘 보너스
저녁 총 예산 = 저녁 기본 식비 + 오늘 보너스
```

v1에서는 점심/저녁에 같은 보너스를 보여줘도 된다.
추후에는 사용자가 보너스를 점심/저녁 중 어디에 쓸지 선택하게 만들 수 있다.

## Focus Timer

### 흐름

1. 활동 선택
2. 시작 버튼
3. 타이머 진행
4. 종료 버튼
5. 결과 화면
6. 오늘 보너스에 반영

### 타이머 정책

- 초 단위로 화면 업데이트
- 보너스 계산은 완료된 분 기준
- 1분 미만은 0원
- 앱 종료/백그라운드 처리는 v1에서는 단순 처리

### 결과 화면

표시 항목:

- 활동명
- 집중 시간
- 획득 금액
- 오늘 누적 보너스
- 현재 저녁 예산
- 홈으로 돌아가기
- 메뉴 추천받기

## Menu Recommendation

### 흐름

1. 식사 선택: 점심/저녁
2. 질문 1: 오늘 느낌
3. 질문 2: 음식 종류
4. 질문 3: 누구와 먹는지
5. 추천 결과 표시

### 결과 표시

- 추천 메뉴 3개
- 예상 가격
- 남는 예산
- 추천 이유
- 예산 티어 문구

예:

- "오늘은 16,000원이니까 든든한 한식 가능!"
- "제육덮밥 + 계란추가 추천"
