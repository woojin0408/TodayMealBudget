# Requirements

## 플랫폼

- iOS 앱
- SwiftUI 사용
- iPhone 우선
- 다크모드 대응은 v1에서는 필수 아님
- 외부 서버 사용하지 않음

## 필수 기능

### 1. 오늘 예산 보기

사용자는 홈 화면에서 오늘의 점심/저녁 예산을 볼 수 있어야 한다.

표시 항목:
- 기본 점심 식비
- 기본 저녁 식비
- 오늘 획득한 집중 보너스
- 총 점심 예산
- 총 저녁 예산

### 2. 집중 타이머

사용자는 활동을 선택하고 집중 타이머를 시작할 수 있어야 한다.

활동 종류:
- 공부
- 코딩
- 독서

기본 보너스:
- 공부: 1분당 100원
- 코딩: 1분당 70원
- 독서: 1분당 50원

### 3. 보너스 계산

집중이 완료되면 활동별 분당 금액에 따라 보너스를 계산한다.

예:
- 공부 30분 x 100원 = 3,000원
- 코딩 60분 x 70원 = 4,200원

초 단위는 버리고 완료된 분 단위로 계산한다.

### 4. 메뉴 추천

사용자는 메뉴 추천 화면에서 3가지 질문에 답한다.

질문:
1. 오늘은 어떤 느낌인가?
2. 어떤 종류가 끌리는가?
3. 누구와 먹는가?

앱은 현재 예산, 선택한 답변, 메뉴 데이터를 바탕으로 추천 메뉴를 보여준다.

### 5. 설정

사용자는 아래 값을 수정할 수 있어야 한다.

- 점심 기본 식비
- 저녁 기본 식비
- 공부 분당 보너스
- 코딩 분당 보너스
- 독서 분당 보너스

## 제외 기능

v1에서는 아래 기능을 만들지 않는다.

- 로그인
- 클라우드 동기화
- 친구 기능
- 실제 결제
- 위치 기반 식당 추천
- 배달앱 연동
- AI API 연동
