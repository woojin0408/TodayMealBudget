# MVP 계획

## 목표

오늘 쓴 밥값을 입력하면 최저시급 10,320원 기준으로 몇 분/몇 시간어치인지 직관적으로 보여준다.

## 핵심 기능

1. 오늘 밥값 입력
2. 고정 기준값 적용
   - 최저시급 10,320원
3. 결과 표시
   - 최저시급 기준 몇 분/몇 시간어치인지
   - 1분당 금액

## 구현 범위

- 기존 React, TypeScript, Vite, Tailwind 기반 구조 유지
- 기존 화면 라우팅 방식에 `몇분치` 페이지 추가
- 입력값은 클라이언트 상태로만 관리
- 계산 로직은 작은 서비스 함수로 분리

## 제외 범위

- 로그인
- 서버 연동
- 카드/은행 연동
- 복잡한 통계
- AI 분석
- 여러 사용자 기능
