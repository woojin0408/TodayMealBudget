# 오늘의 밥값 v1

집중한 시간만큼 오늘 점심/저녁 식비 보너스를 쌓고, 예산에 맞는 메뉴를 추천해주는 모바일 퍼스트 웹앱.

## 기술 스택

- React
- Vite
- TypeScript
- Tailwind CSS
- localStorage

## 주요 기능

- 오늘 점심/저녁 예산 보기
- 공부/코딩/독서 타이머
- 집중 시간만큼 보너스 식비 증가
- 질문 3개 기반 메뉴 추천
- 금액대별 메뉴 리스트
- 설정에서 기본 식비와 분당 보너스 금액 수정
- localStorage 저장

## 실행 방법

```bash
npm install
npm run dev
```

## 빌드

```bash
npm run build
```

## 제외 기능

- 서버
- 로그인
- 결제
- 위치 기반 추천
- 실제 배달앱 연동
